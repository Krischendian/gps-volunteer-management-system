# GPS Volunteer Management System

> Live: gps-volunteer-system.vercel.app  
> Built for Gifted People Services — registered Ontario 
> non-profit, est. 2011, serving 1,200+ families  
> Actively used by 50–100 volunteers

Full-stack volunteer coordination platform replacing all 
paper-based processes. Delivered solo in 2 weeks (July 2025).

## What it does
- 7-step digital onboarding with e-signature and 
  policy agreements
- Volunteer hours logging with auto-calculated duration
- Event sign-up with real-time capacity tracking
- Points-based tier system (Bronze → Platinum) for 
  annual best-volunteer recognition
- Manager dashboard with org-wide impact metrics 
  and full audit log
- One-click HR reference letter lookup

## Tech stack
React 19 · TypeScript · Firebase Auth · Firestore · 
Google Gemini API · Tailwind CSS · Recharts · Vercel
